# Selling Items Database Project

##  Overview

This is a simple and focused database-based project designed to **add selling items** and store their details securely. The application allows users to insert item information into a database in a structured manner. The main goal of this project is to practice backend logic and database integration.

---

##  Features

* Add selling items to the database
* Store item details such as name, price, and quantity
* Simple and clean database structure
* Beginner-friendly project

---

##  Tech Stack

* Backend: Node.js , Express
* Database: MySQL
* Frontend: HTML

---

##  Project Use Case

This project is useful for understanding:

* Database connectivity
* Backend form handling
* Data storage concepts

---

##  How It Works

1. User enters selling item details
2. Data is sent to the backend
3. Backend validates the data
4. Item details are stored in the database
